Oh snap! Your browser is not supported.

How do I fix this?

Go to browsehappy.com to upgrade your browser to the latest version.

Why do I keep seeing this message?

You are using an unsupported browser. This means your web browser is either too old or the device you are using to access MoDeCaSo is not supported. MoDeCaSo uses advanced and modern web frameworks which require a more modern browser in order to run properly.

Which browser versions support MoDeCaSo?

The following web browsers and versions have been tested to work with MoDeCaSo:

  • Windows Internet Explorer, Version 10 and above
  • Mozilla Firefox, Version 29 and above
  • Google Chrome, Version 38 and above
  • Opera, Version 27.0 and above
  • Apple Safari, Version 5.1.7 and above

Please make sure you update to the newest version of your web browser for the best user experience.

Note: This MoDeCaSo Web Frontend does not run on mobile devices, such as smartphones or tablets.
